## 🤖 AI-Chatbot Project

The **AI-Chatbot** project is a web-based conversational application built using **React** and **Vite**. It allows users to type prompts and receive AI-generated responses through the **Google Gemini API**. The project demonstrates how frontend development can be connected with machine learning-based text generation.

### Highlights

* User-friendly chat interface
* Real-time response generation
* Loading state for better user experience
* Clean integration between frontend and AI API

### Purpose of the Chatbot

This chatbot project is designed to showcase the practical use of AI in modern applications. It helps users understand how natural language input can be processed and turned into meaningful responses using external AI services.
