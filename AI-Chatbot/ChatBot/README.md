# AI-Chatbot

This project is a simple AI-powered chatbot built with **React** and **Vite**. It allows users to enter text prompts and receive responses generated by the **Google Gemini API**.

## Project Overview

The chatbot provides a clean and interactive interface where users can:

* Type questions or prompts
* Send requests to the AI model
* View responses in real time
* Experience a loading state while the answer is being generated

## Technologies Used

* React
* Vite
* JavaScript
* Google Generative AI SDK
* CSS for styling

## How It Works

1. The user enters a prompt in the chat input box.
2. The frontend sends the prompt to the Gemini API.
3. The API returns a response.
4. The chatbot displays the response on the screen.

## Purpose

This project demonstrates how AI can be integrated into a web application to provide intelligent and helpful responses in a user-friendly way.
